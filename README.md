# 🎓 Campus Skill & Placement Readiness Tracker

A full-stack Django web application that helps students track 
their skills, calculate placement readiness score, identify 
skill gaps, and get career recommendations.

## 🛠️ Tech Stack
- Python · Django · SQLite3
- HTML · CSS · Bootstrap 5 · JavaScript · Chart.js

## ✨ Key Features
- Placement Readiness Score (out of 100)
- Skill Gap Analyzer by target role
- Career Recommendation System
- Resume Upload & PDF Keyword Analyzer
- MCQ Quiz Module
- Activity Dashboard

## ▶️ Run Locally
git clone https://github.com/YourUsername/placement-readiness-tracker.git
cd placement-readiness-tracker
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
